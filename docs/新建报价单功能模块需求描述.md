# 报价单表-基本信息

- 报价单编号
   编码规则：BJ-年月日8位数-三位序号（补零）；如：BJ-20260611-001
   自动生成：查询数据库报价单表（用“BJ-今日年月日8位”搜索）-搜到降序第一个-提取
   
- 【搜选】联系人
   若选项没有，可以选择“新增填写”，变为输入框

- 客户公司名称
- 职位
   以上三个，搜索联系人选择即可，自动填写另两个
   若联系人选了“新增填写”，则变为输入框待手动填写
   
- 市场部经办人
   根据登录用户自动填写
   
- 报价日期
   可由系统自动生成当天日期，显示为年月日，实际值为时间戳
   
- 【点选】报价类型
   选择项：A / B / C，【根据选择，展开A/B表单】

点击选项A或B时，下方展开A或B的明细，并且有来新加明细（）

点击提交时，将填写数据整理好，调用后端Gin的接口传入（先写为样板，后端接口尚未实现）
